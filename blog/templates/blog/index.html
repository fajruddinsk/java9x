Hibernate

Hibernate is light weight,non-invasive and open source framework (ORM-tool) that is allows to devlop Data Base portal, Data Base independent persistence logic with out converting Object to text or simple value

Spring

Spring is an opensource,light weight,loosly coupled aspect oriented,dependency injection based java application framework to devlop various java applications(All kinds of application Desktop app,Web app,enterprise application) .

Programming language

It is directly installed software having features to develop software application. It is defines syntaxes(Rules),semantics(structures) of programing.These are based to create software technologies framework, tools, DB Software, Operating Systems and etc.. These are raw materials of software programing.Eg:C,C++,C#,Java

Software Technology

It is software specification having set of rules and guidelines to develop software by using programing language support. Software technology is not installable but software created based on software technologies is installable. Working with software is nothing but working with software technologies. In java environment, technology rule are nothing but interfaces and guideline are nothing but classes. Abstract class represent both rules and guidelines. Eg: JDBC, JSP, Servlet and etc. JDBC technology gives rules and guidelines to develop driver using java language. Working with JDBC Driver is nothing but working with JDBC technology

There are 2 type of software technologies

    1Open Technologies

    2Proprietary technologies

Open Technologies:

Here the rules and guidelines of technology are open to all software vendor companies to develop software. Ex: All JAVA Technologies (JDBC, Servlet, JSP etc.)

Proprietary technologies:

Here the vendor company who has given technology is only allowed to develop software based on technologies. Ex: All Microsoft Technologies (ASP.NET)

Framework

Framework is not a installable software that uses existing technologies internally to simplify the application development having ability to generate common logic of application.Framework is not installable software that provides abstraction layer on existing technologies to simplify the application development process. Ex: Struts, Spring, Hibernate.
